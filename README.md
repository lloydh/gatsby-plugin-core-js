# gatsby-plugin-core-js

Polyfill gatsby with core-js

## Install

`yarn add gatsby-plugin-core-js`

## How to use

Add the plugin to your `gatsby-config.js`.

```js
plugins: ['gatsby-plugin-core-js']
```
